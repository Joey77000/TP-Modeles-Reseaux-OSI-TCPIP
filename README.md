# TP Modèle OSI et TCP/IP

## Objectif

Ce dépôt regroupe plusieurs travaux pratiques autour du modèle OSI, du modèle TCP/IP et de l’analyse des communications réseau.

L’objectif est de comprendre le rôle de chaque couche, les protocoles associés et la manière dont les données circulent dans un réseau.

## Notions étudiées

- Modèle OSI
- Modèle TCP/IP
- Encapsulation des données
- Adressage MAC et IP
- Protocoles TCP et UDP
- Services applicatifs
- Diagnostic réseau
- Analyse de trames

## TP présents

- Présentation du modèle OSI
- Couche 1 : physique
- Couche 2 : liaison de données
- Couche 3 : réseau
- Couche 4 : transport
- Couches 5, 6 et 7 : session, présentation, application
- Encapsulation
- Analyse réseau avec Wireshark

## Compétences mobilisées

- Compréhension des modèles réseau
- Analyse de protocoles
- Diagnostic réseau
- Lecture de trames
- Utilisation de Wireshark
- Bases TCP/IP
